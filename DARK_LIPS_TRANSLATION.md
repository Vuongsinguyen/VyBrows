## Ghi chú

- File services.json cần cập nhật template code để hỗ trợ các field đa ngôn ngữ mới
- Các service khác cũng cần được cập nhật tương tự
- Template `/src/pages/[lang]/service/[service].astro` cần điều chỉnh để đọc các field đa ngôn ngữ
